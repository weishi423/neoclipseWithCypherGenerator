Introduction

What is Neoclipse?

Neoclipse is a tool for visualizing and altering Neo4j databases, including nodes, relationships and properties on both. The purpose is to support the development of Neo4j applications.

In the Database graph view, it is possible to edit properties of both nodes and relationships. Further you can add new relationship types, relationships and nodes. There are also some ways to decorate the nodes and relationship representations.

How to use Neoclipse?

You can run Neoclipse both as a standalone product or as a plug-in for Eclipse IDE 3.4+.

For details on running Neoclipse, please refer to the basic tutorial.
